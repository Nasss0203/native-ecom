## 1. Navigation & Screen Management

| Package                          | Mô tả                                           |
| -------------------------------- | ----------------------------------------------- |
| `@react-navigation/native`       | Thư viện điều hướng chính trong React Native    |
| `@react-navigation/native-stack` | Điều hướng dạng stack (màn hình chồng lên nhau) |
| `react-native-gesture-handler`   | Hỗ trợ thao tác vuốt, chạm, kéo                 |
| `react-native-reanimated`        | Tạo hiệu ứng chuyển động mượt và tối ưu         |
| `react-native-screens`           | Quản lý màn hình native hiệu quả                |
| `react-native-safe-area-context` | Xử lý vùng tai thỏ / notch và thanh trạng thái  |
| `@react-navigation/bottom-tabs`  |                                                 |
| `npm i react-native-worklets`    |                                                 |

## 2. UI & Icons

| Package                          | Mô tả                                                |
| -------------------------------- | ---------------------------------------------------- |
| `react-native-vector-icons`      | Bộ icon phổ biến (FontAwesome, Material Icons, v.v.) |
| `react-native-status-bar-height` | Lấy chiều cao thanh trạng thái để bố trí giao diện   |
| `react-native-bootsplash`        | Tạo màn hình khởi động (Splash Screen) chuyên nghiệp |

## 3. State Management & Logic

| Package                                     | Mô tả                                       |
| ------------------------------------------- | ------------------------------------------- |
| `@reduxjs/toolkit`                          | Cấu trúc Redux đơn giản và tối ưu hiệu suất |
| `react-redux`                               | Kết nối Redux với React Native              |
| `@react-native-async-storage/async-storage` | Lưu dữ liệu cục bộ (token, cache, v.v.)     |

## 4. Networking & API

| Package               | Mô tả                                                        |
| --------------------- | ------------------------------------------------------------ |
| `axios`               | Gọi API HTTP, quản lý request dễ dàng                        |
| `react-native-dotenv` | Dùng file `.env` để lưu biến môi trường (API_URL, KEY, v.v.) |
